Animações e Efeitos Globais

O fyLib possui um motor de movimento poderoso que desacopla a lógica visual da interação de UI.

Como Funciona?

O pacote @fylib/animation gerencia dois tipos de movimentos:

Microinterações

Animações locais de componentes, como hover em botões, foco em inputs ou entrada de cards. Elas são definidas por tema mas podem ser sobrescritas por instância.

Efeitos Globais

Efeitos que afetam a tela toda ou regiões amplas, como confetes, flashes de janela ou transições de sidebar. São disparados por eventos de UI mapeados no AppConfig.

Efeitos Disponíveis

Estes efeitos podem ser disparados centralizadamente via effectTriggers no config ou manualmente através das propriedades de componente.

Nome do Efeito Descrição
confetti Dispara uma chuva de confetes coloridos sobre a tela. Aceita parâmetros de intensity, speed e loop. Requer que effectsEnabled esteja ativo no config.
hearts Uma chuva de corações animados utilizando as cores do tema Puffy. Ideal para momentos de sucesso ou temas românticos. Regra de Ativação: Requer themeEffectsEnabled: true no config E a presença da prop bgEffect no componente layout.
snow Efeito de neve caindo, ideal para o tema de Natal. Aceita intensity e speed.
bubbles Bolhas flutuantes subindo, ideal para o tema Windows XP.
matrix Chuva digital de caracteres verdes, ideal para o tema Nexus Neon.
particles Partículas flutuantes sutis, ideal para temas limpos e modernos como o Default.
stars Estrelas cintilantes ao fundo, ideal para o tema Workbench 3.
dots Padrão técnico de pontos em movimento, ideal para o tema Finey Hub.
window-open Um flash suave branco simulando a abertura de uma nova janela.
sidebar-slide-in Glow azul na borda lateral simulando a entrada do menu.

Disparando Efeitos via Eventos

Você pode mapear eventos de UI para efeitos globais centralizadamente no seu AppConfig:

export const themeConfig: AppConfig = {
  theme: 'default',
  effectTriggers: {
    'fy-button.click': 'confetti',
    'fy-card.submit': 'confetti',
    'fy-layout.enter': 'window-open'
  }
};
