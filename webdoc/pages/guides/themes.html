Sistema de Temas

Baseado em Design Tokens, o fyLib permite customização total da interface, desde cores atômicas até comportamentos complexos.

Temas Disponíveis

Default

Moderno, limpo e profissional. Focado em produtividade.

Windows XP

Nostalgia pura. Gradientes fortes e bordas clássicas da era Luna.

Workbench (macOS)

Inspirado no macOS. Glassmorphism, transparências e sombras suaves.

Windows 7

Estética Aero. Efeitos de vidro e transparências azuladas.

Finey Puffy

Lúdico e adorável. Tons de rosa, bordas extra arredondadas e efeitos de corações.

Finey Hub

Inspirado na UI do GitHub. Clean, profissional e com toques sutis de elementos 8-bit.

Finey Nexus

Focado em alta tecnologia e conexões, com estética baseada em grids cibernéticos neon.

Recursos Especiais do Tema

Além de tokens visuais, os temas no fyLib podem carregar comportamentos imersivos:

Background Effects

Efeitos dinâmicos (como corações caindo) que são disparados automaticamente ao carregar o tema. Regra de Ativação: Requer themeEffectsEnabled: true no config E a presença da prop bgEffect no componente.

Wallpapers

Padrões de fundo animados (SVG) que trazem textura e movimento constante. Regra de Ativação: Requer wallpaperEnabled: true no config E a presença da diretiva fyWallpaper no elemento HTML.

O que são Design Tokens?

No fyLib, tokens são variáveis semânticas que abstraem valores brutos (como #3b82f6) para nomes com propósito (como colors.primary).

Exemplo de Token no Tema
tokens: {
  colors: {
    primary: '#22c55e',
    background: '#0a0f12',
    text: '#eafaf2'
  },
  effects: {
    card: {
      shadow: '0 24px 50px rgba(0,0,0,0.6)',
      borderColor: 'rgba(34, 197, 94, 0.7)'
    }
  }
}

Suporte a Light & Dark Mode

Todos os temas do fyLib possuem suporte nativo a modo escuro. O engine mescla automaticamente os darkTokens sobre os tokens base quando o modo é alterado.

Modo Light

Tokens base aplicados para máxima legibilidade em ambientes claros.

Modo Dark

Overrides automáticos para superfícies escuras e baixo contraste visual.
