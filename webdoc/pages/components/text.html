Text (Crypto)

Componente utilitário para renderização de texto com suporte nativo a descriptografia em tempo real (AES-GCM).

Prévia Visual

Texto Criptografado (no banco):
U2FsdGVkX1+VnJpZ... (hash seguro)
Renderizado via fy-text:
Informação Confidencial

Exemplo de Uso

O componente utiliza o cryptoEngine do fyLib para descriptografar dados sensíveis de forma transparente se configurado.

<!-- Texto Simples -->
<fy-text text="Olá Mundo"></fy-text>

<!-- Texto Criptografado -->
<fy-text 
  [text]="encryptedHash" 
  [cryptoEnabled]="true">
</fy-text>

Propriedades (Inputs)

Nome Tipo Default Descrição
textstring''Conteúdo textual ou hash criptografado.
cryptoEnabledbooleanfalseHabilita a tentativa de descriptografia automática.

Configuração Necessária

Para o modo crypto funcionar, você deve fornecer a chave e o IV no AppConfig:

provideFyLib({
  crypto: {
    enabled: true,
    key: 'sua-chave-secreta-32-chars',
    iv: 'seu-iv-12-chars'
  }
})
