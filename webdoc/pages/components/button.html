React (Em breve)

Button

Componente versátil de ação com suporte nativo a temas, animações e efeitos globais.

Prévia Interativa

Primary
Secondary
Ghost
Delete

Exemplo de Uso

<!-- Botão Simples -->
<fy-button variant="primary">Salvar</fy-button>

<!-- Com Ícone e Efeito de Confete -->
<fy-button 
  variant="success" 
  iconName="check" 
  [clickEffect]="'confetti'">
  Finalizar Compra
</fy-button>

Propriedades (Inputs)

Nome Tipo Default Descrição
variant 'primary' | 'secondary' | 'ghost' | 'danger' 'primary' Estilo visual do botão definido pelo tema.
iconName string undefined Nome do ícone Phosphor para exibir.
clickEffect EffectName undefined Efeito global disparado ao clicar (ex: 'confetti').

Design Tokens (Efeitos)

O comportamento visual do botão é controlado pelo grupo effects.button no tema.

Token Descrição
backgroundCor de fundo ou gradiente.
borderColorCor da borda.
shadowSombra da caixa (elevation).
textColorCor do texto interna.
