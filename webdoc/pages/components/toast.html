Toast

Sistema de notificações temporárias para feedback rápido ao usuário.

Prévia Visual

Sucesso
Operação realizada com sucesso!

Exemplo de Uso

O Toast geralmente é disparado via serviço (FyNotificationService):

import { FyNotificationService } from '@fylib/adapter-angular';

@Component({...})
export class MyComponent {
  constructor(private notify: FyNotificationService) {}

  showSuccess() {
    this.notify.success({
      title: 'Salvo!',
      message: 'As alterações foram aplicadas.'
    });
  }
}

Métodos do Serviço

Método Descrição
success(config)Notificação de sucesso (verde).
error(config)Notificação de erro (vermelho).
warning(config)Notificação de aviso (amarelo).
info(config)Notificação informativa (azul).

Design Tokens (Efeitos)

O grupo effects.toast permite customização extrema por tema.

Token Descrição
backgroundFundo da notificação.
iconColor.info|success|...Cores específicas dos ícones por tipo.
icons.info|success|...Mapeamento de quais ícones usar por tipo.
shadowSombra da notificação.
