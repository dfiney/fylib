Select

Componente de seleção robusto com suporte a busca, múltipla escolha e tematização avançada.

Prévia Visual

Escolha uma opção
Angular Framework
React Library
Vue.js Ecosystem

Exemplo de Uso

<!-- Select Simples -->
<fy-select 
  [options]="frameworks" 
  placeholder="Selecione o framework"
  [(value)]="selected">
</fy-select>

<!-- Com Busca e Múltipla Escolha -->
<fy-select 
  [options]="tags" 
  [searchable]="true" 
  [showCheckbox]="true"
  [closeOnSelect]="false"
  [value]="['tag-1', 'tag-2']">
</fy-select>

Propriedades (Inputs)

Nome Tipo Default Descrição
options{ label, value }[][]Lista de opções disponíveis.
valuestring | string[]undefinedValor(es) selecionado(s).
searchablebooleanfalseHabilita barra de busca interna.
showCheckboxbooleanfalseExibe checkboxes nas opções (útil para multi-select).
size'sm' | 'md' | 'lg''md'Tamanho visual do controle.
