Table

Componente robusto para exibição de dados tabulares com suporte a ordenação, paginação e temas.

Prévia Visual

ID Usuário Status
1Victor BarberinoAtivo

Exemplo de Uso

<fy-table [data]="users" [columns]="columns" [striped]="true">
  <!-- Template Customizado para Coluna -->
  <ng-template fyColumn="status" let-user>
    <span [class]="user.active ? 'text-success' : 'text-danger'">
      {{ user.active ? 'Online' : 'Offline' }}
    </span>
  </ng-template>
</fy-table>
