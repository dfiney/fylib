React (Em breve)

Input

Campo de entrada de texto altamente customizável com suporte a estados, ícones e validações.

Prévia Visual

Exemplo de Uso

<!-- Input Simples -->
<fy-input label="E-mail" placeholder="seu@email.com"></fy-input>

<!-- Com Ícone e Status de Erro -->
<fy-input 
  label="Senha" 
  type="password"
  iconName="lock"
  status="error"
  errorMessage="A senha é obrigatória">
</fy-input>
